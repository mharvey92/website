Project Description

Image Credit: Ozone.com

This website is part of an individual assignment from the class "Unleashing Open Data with Python" at Johns Hopkins.

The goal of this project is for each student to pick a research topic and then, using the programming language Python, scrape related data from APIs, websites, and csv files. After the data in the methodology section is accessed, statistical analysis will be implemented to help answer the research question: What countries, states, and industries, emit the largest amounts of greenhouse gases? This question will be communicated in tables and data visualizations created in R and Python on the Results page.

This website was built by Mariah Harvey using Flask, Jinja2, and additional HTML/CSS code to display this project.
